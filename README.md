# Eventful Journey Hub

Web app project focused on event and journey-style user experiences.

## Purpose

This repository is used to design and iterate product flows, UI, and content around event/journey concepts.

## Expected Experience

- Fast local development loop
- Structured UI experimentation and validation
- Preview deployment for stakeholder review

## Getting Started

```bash
npm install
npm run dev
```

## Notes

Current stack is front-end oriented and suited for rapid iteration.
